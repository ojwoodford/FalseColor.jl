FalseColor.jl
=============

The aim of this package is to provide tools to convert matrices of data into nice looking images that can then be saved using imwrite.

I will start by copying the functionality in two of my MATLAB packages: [real2rgb](http://www.mathworks.com/matlabcentral/fileexchange/23342) and [sc](http://www.mathworks.com/matlabcentral/fileexchange/16233).
